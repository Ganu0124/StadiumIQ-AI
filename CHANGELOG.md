# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-07-10

### Added
- Reusable UI component library (Button, Card, Badge, Modal, Tabs, Tooltip, Progress, Skeleton loader).
- 16 interactive dashboard pages (Executive, Map, Crowd, Operations, Incidents, Security, Medical, Parking, Food, Concierge, Announcements, AI Recommendations, Analytics, Reports, Settings, Profile).
- Google Gemini decision support prompts for crowd, threat, and restock playbooks.
- Strict TypeScript types and constants for all operational domains.
- Full setup assets: Dockerfile, docker-compose, and GitHub Actions.
